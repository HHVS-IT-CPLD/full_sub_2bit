# 2 位元減法器（2 的補數法）

## 1. 專案簡介

本專案使用 Verilog 實作一個 2 位元減法器，運算目標為：

\[
D = A - B
\]

其中：

- `A`、`B` 為 2 位元無號輸入。
- `D` 為 2 位元差值輸出。
- `Borrow` 為借位輸出（當 `A < B` 時為 1）。

設計核心採用 **2 的補數** 方法，將減法轉換為加法：

\[
A - B = A + (\sim B + 1)
\]

## 4. 程式架構與設計流程

程式位於 `full_sub_2bit.v`，邏輯可拆成 4 個層次：

### 4.1 輸入擴位（Zero Extension）

為了保留最高位進位資訊，先把 2 位元輸入擴成 3 位元：

- `{1'b0, A}`
- `{1'b0, (...)}`

這樣在做加法時，`sum_temp[2]` 就能記錄最終進位。

### 4.2 產生減數的 2 的補數

對 `B` 做下列轉換：

1. 取反：`~B`
2. 加一：`~B + 2'b01`

得到 `B` 的 2 的補數。

### 4.3 以加法實現減法

使用以下組合邏輯：

```verilog
assign sum_temp = {1'b0, A} + {1'b0, (~B + 2'b01)};
```

其中 `sum_temp` 為 3 位元暫存結果。

### 4.4 輸出映射

1. 差值輸出：

```verilog
assign D = sum_temp[1:0];
```

2. 借位輸出：

```verilog
assign Borrow = ~sum_temp[2];
```

說明：

- 在 2 的補數減法中，若最高位有進位（`sum_temp[2] = 1`），代表不需借位。
- 若最高位無進位（`sum_temp[2] = 0`），代表有借位。
- 因此借位旗標為最高位進位反相。

---

## 5. 關鍵程式碼（完整模組）

```verilog
module full_sub_2bit (
    input  [1:0] A,
    input  [1:0] B,
    output [1:0] D,
    output       Borrow
);

    // 使用 3 位元暫存加法結果：最高位用來判斷是否借位
    wire [2:0] sum_temp;

    // 2 的補數減法：A - B = A + (~B + 1)
    assign sum_temp = {1'b0, A} + {1'b0, (~B + 2'b01)};

    // 低 2 位元即為差值
    assign D = sum_temp[1:0];

    // 最高位進位為 0 代表發生借位，因此 Borrow = ~CarryOut
    assign Borrow = ~sum_temp[2];

endmodule
```

## 6. 運算範例

| A (bin) | B (bin) | 十進位運算 | D (bin) | Borrow |
|---:|---:|---:|---:|---:|
| 10 | 01 | 2 - 1 = 1 | 01 | 0 |
| 01 | 10 | 1 - 2 = -1（2 位元為 11） | 11 | 1 |
| 11 | 01 | 3 - 1 = 2 | 10 | 0 |
| 00 | 01 | 0 - 1 = -1（2 位元為 11） | 11 | 1 |

> 備註：2 位元輸出僅保留最低 2 位，因此負數會以 2 的補數形式表示。

---

## 7. 設計特性

- 全組合邏輯（`assign`），無時脈需求。
- 使用單一加法器概念實現減法，電路結構清楚。
- 易於擴充成 N 位元減法器（調整位元寬度即可）。
